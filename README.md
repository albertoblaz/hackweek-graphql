# HackWeek (GraphQL + Electron)

## How to install

1. Run server:
   ```bash
   yarn
   yarn start
   ```

2. Run client on web (on another console tab):
   ```bash
   cd client
   yarn
   yarn start
   ```
   (if it does not work, reload page)

3. Run desktop client with Electron (from `client/`)
   `yarn electron`
